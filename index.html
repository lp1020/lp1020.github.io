_______ PROJECTS _______

CRUELTY-FREE-4-ME

CrueltyFree4Me is a community listing of cruelty-free and non-cruelty free companies, as well as a location tracker and vegetarian restaurant finder with user authentication, geolocation and maps. It is built with Ruby on Rails, ERB, React, CoffeeScript, PostgreSQL, and the Google Places API.

MUSIC SPACE

Music Space is a full CRUD Ruby on Rails application along with ERB, CSS, and PostgreSQL for keeping track of your favorite artists and songs. Users can add, edit, or delete their favorite artists as well as add, edit, or delete their favorite songs by those artists. Users can also display a YouTube embedded music video of each song.

FIND-A-FANTASTIC-BEAST

This is a full CRUD petfinder application for magical (and non-magical) animals using postgreSQL, Node.js/Express, and ReactJS with user authentication. It uses the Yelp API to display a real local animal shelter from each borough in New York City. It then allows the user to view, add, edit, or "adopt" (aka delete) available fake animals from the database.

KITTY HEALTH TRACKER

This is a full CRUD health tracker app which tracks daily health stats and displays users' average stats. It is built using Node.js/Express, postgreSQL, and EJS. It employs the GIPHY API to display funny cat GIFs throughout the site as an added fun touch.

JEOPARDY!

The classic trivia game show in which players answer questions with different monetary values. If users answer the question correctly, money is added to their bank, and if they are incorrect, it is subtracted. It is built with HTML, CSS, JavaScript, and jQuery.

OFF TO THE RACES

This is a two-player race car game in which each car is assigned a key on the keyboard in order to move it. The player who gets their car across the finish line first wins. Built with JavaScript, HTML, and CSS.
